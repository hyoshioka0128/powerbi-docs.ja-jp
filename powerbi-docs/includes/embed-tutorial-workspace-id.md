ワークスペース ID の GUID を取得するには、これらの手順に従います。

1. Power BI サービスにサインインします。

2. 埋め込むレポートを開きます。

3. URL から GUID をコピーします。 GUID は **/groups/** と **/reports/** の間にある数値です。
