Power BI により、レポート、ダッシュボード、およびタイルがワークスペースに保持されます。 これらのアイテムを埋め込むには、それらを作成してワークスペースにアップロードする必要があります。

>[!TIP]
>既にワークスペースがある場合は、この手順を省略できます。

ワークスペースを作成するには、次のようにします。

1. Power BI にサインインします。

2. **[ワークスペース]** を選択します。

3. **[ワークスペースの作成]** を選択します。

4. ワークスペースに名前を付けて、 **[保存]** を選択します。
